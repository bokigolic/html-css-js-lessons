Lesson 6: HTML Lists

📋 Why Use Lists?

Lists help organize content. You can use them for menus, steps, features, definitions, and more. HTML supports several types of lists to suit different needs.

🔹 Unordered List – <ul>

Used for lists without a specific order (e.g. shopping list):

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

🔸 Ordered List – <ol>

Used when order matters (e.g. recipe steps):

<ol>
  <li>Preheat oven</li>
  <li>Mix ingredients</li>
  <li>Bake for 30 minutes</li>
</ol>

🎨 Styling Lists

You can change the look with CSS:

ul {
  list-style-type: square;
}
ol {
  list-style-type: upper-roman;
}

📦 Nested Lists

You can nest one list inside another for sub-items:

<ul>
  <li>Fruits
    <ul>
      <li>Apples</li>
      <li>Bananas</li>
    </ul>
  </li>
  <li>Vegetables</li>
</ul>

📖 Description List – <dl>

Used for terms and definitions (like a glossary):

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

🧠 Practice Activity

Create a weekly to-do list with:

  • <ol> for weekday tasks
  • <ul> for weekend fun
  • <dl> for short glossary

📌 Summary

  • <ul> – unordered (bulleted) list
  • <ol> – ordered (numbered) list
  • <li> – list item inside ul/ol
  • <dl>, <dt>, <dd> – description list
Mini Challenge: Create a recipe card using an ordered list for steps and an unordered list for ingredients.
