🎯 Final Mini Project: Build Your Own Web Page

Your task is to create a basic web page that represents something meaningful to you: your bio, your favorite topic, or a personal project.

✅ Requirements:

  • Use <header>, <main>, and <footer>
  • Include a navigation menu with <a> links
  • Add at least 1 image and 1 list
  • Include a form (contact or signup)
  • Use semantic elements: <section>, <article>, <aside>

🧰 Starter Template (You Can Copy This)

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Personal Page</title>
</head>
<body>
  <header>
    <h1>Welcome to My Web Page</h1>
  </header>
  
  <nav>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
  </nav>

  <main>
    <section id="about">
      <h2>About Me</h2>
      <p>I love building websites and learning HTML!</p>
      <img src="me.jpg" alt="A picture of me" />
    </section>

    <section>
      <h2>My Favorite Things</h2>
      <ul>
        <li>Coding</li>
        <li>Design</li>
        <li>Coffee</li>
      </ul>
    </section>

    <section id="contact">
      <h2>Contact Me</h2>
      <form>
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required />
        <label for="message">Message:</label>
        <textarea id="message" name="message" rows="5"></textarea>
        <button type="submit">Send</button>
      </form>
    </section>
  </main>

  <footer>
    <p>© 2025 MySite. All rights reserved.</p>
  </footer>
</body>
</html>
