Lesson 3: Elements and Attributes

HTML Elements – The Core Building Blocks

In HTML, an element is everything from the opening tag to the closing tag. Elements define the structure and content of your page. Almost everything in HTML is an element. 

<p>This is a paragraph.</p>

Here, <p> is the element. It starts with an opening tag, contains content, and ends with a closing tag.

💡 Tip: Not all elements have a closing tag! Some are self-closing, like <br> or <img>.

📦 Nested Elements

HTML elements can be placed inside other elements. This is called nesting.

<div>
  <h2>Title</h2>
  <p>This is a <strong>nested</strong> paragraph.</p>
</div>

The <strong> tag is nested inside the <p> tag, which is inside the <div>.

⚠️ Rule: Always properly close and nest your tags to avoid rendering issues.

🔑 What Are Attributes?

Attributes provide extra information about an element. They're always included in the opening tag and usually come in name="value" pairs. 

<a href="https://example.com" target="_blank">Visit Example</a>
  • href – specifies the link URL
  • target – determines how the link opens (e.g. new tab)

🌐 Global Attributes

These attributes can be used on almost any HTML element:

  • id – Unique identifier for an element
  • class – Category name used in CSS or JavaScript
  • style – Inline styling (e.g. style="color:red")
  • title – Tooltip text shown on hover
<p id="intro" class="highlight" style="color:blue" title="Intro paragraph">
  Welcome to my page!
</p>

🧪 Practice Activity

Write an HTML snippet that uses:

  • an <img> tag with src and alt
  • a <p> with class and title
<img src="cat.jpg" alt="A cute cat looking curious">
<p class="caption" title="Cat photo description">Meet my cat Miso!</p>

📌 Summary

  • HTML elements define structure. They usually have an opening and closing tag.
  • Elements can be nested to organize content.
  • Attributes provide extra data and control behavior or styling.
  • Global attributes like id, class, and title can be used across elements.
