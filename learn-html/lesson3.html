📘 HTML Basic Examples

In this chapter, we’ll walk through the most common and essential building blocks of an HTML page. Don't worry if some tags are new — everything will make sense step by step.

📄 HTML Document Structure

Every HTML file starts with a declaration that tells the browser: “Hey, I’m writing HTML5!”

<!DOCTYPE html>
<html>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>
  • <!DOCTYPE html> – Tells the browser you're using HTML5.
  • <html> ... </html> – Wraps the whole page.
  • <body> ... </body> – Contains everything visible on the page.

🏷️ Headings

Headings define titles and subtitles. There are six levels:

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

<h1> is the biggest and most important, while <h6> is the smallest.

📑 Paragraphs

Paragraphs are wrapped in the <p> tag:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

🔗 Links

Links are created using the <a> tag and require the href attribute:

<a href="https://www.w3schools.com">Visit W3Schools</a>

The href defines the URL that the link goes to.

🖼️ Images

To show an image, use the <img> tag with some helpful attributes:

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
  • src – The image path or URL.
  • alt – Describes the image if it can’t load or for screen readers.
  • width & height – Controls the image size.

👀 How to View HTML Code in the Browser

1. View Page Source

To see the raw HTML of any page:

  • Windows: Press CTRL + U
  • Mac: Press Command + Option + U
  • Or right-click anywhere on the page and select "View Page Source".

2. Inspect Elements

Want to peek into how something is built?

  • Right-click on any element and choose “Inspect”.
  • You’ll see both the HTML and the CSS!
  • You can even experiment by editing code directly in the panel.
