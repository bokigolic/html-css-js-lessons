Lesson 8: HTML Forms

📨 What Are Forms?

Forms are used to collect user input: login credentials, comments, survey answers, and more. They are essential for interactivity on the web.

🧾 Basic Form Structure

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" />
  <button type="submit">Submit</button>
</form>

🔢 Common Input Types

  • text – single-line input
  • password – hides typed text
  • email – validates email format
  • checkbox – multiple selections
  • radio – single choice from a group
  • submit – submit the form
<input type="text" name="username" />
<input type="password" name="pwd" />
<input type="email" name="email" />
<input type="checkbox" name="subscribe" />
<input type="radio" name="gender" value="male" /> Male
<input type="radio" name="gender" value="female" /> Female

📝 Labels and Textareas

Always associate labels with inputs using the for attribute.

<label for="bio">Your bio:</label>
<textarea id="bio" name="bio" rows="4" cols="30"></textarea>

📋 Dropdown Menus with <select>

<label for="country">Country:</label>
<select id="country" name="country">
  <option value="us">United States</option>
  <option value="de">Germany</option>
  <option value="ba">Bosnia & Herzegovina</option>
</select>

🔘 Buttons

<button type="submit">Submit Form</button>
<button type="reset">Reset</button>

🧪 Example: Feedback Form

<form action="/submit" method="post">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required />

  <label for="msg">Message:</label>
  <textarea id="msg" name="message" rows="5"></textarea>

  <button type="submit">Send Feedback</button>
</form>

📌 Summary

  • <form> groups inputs together
  • Inputs like text, email, radio, and checkbox collect various types of data
  • Always label inputs for accessibility
  • <textarea> for longer text, <select> for dropdowns
