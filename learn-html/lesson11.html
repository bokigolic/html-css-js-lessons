💬 HTML Comments

HTML comments are not shown on the web page, but they are very useful when writing or reviewing code.

🧠 Why Use Comments?

  • To leave notes and reminders in your code
  • To hide sections of HTML for testing or debugging
  • To improve readability for yourself and others

🛠️ Syntax of an HTML Comment

Here’s how you write a comment in HTML:

<!-- Write your comments here -->

✅ Notice:

  • It starts with <!--
  • It ends with -->
  • No exclamation point in the closing tag!

📝 Example: Adding Comments

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

🙈 Hide HTML Content

You can use comments to temporarily hide parts of your HTML code:

<p>This is a paragraph.</p>

<!-- <p>This is another paragraph</p> -->

<p>This is a paragraph too.</p>

You can also hide multiple lines:

<p>This is visible.</p>
<!--
  <p>Look at this cool image:</p>
  <img src="pic_trulli.jpg" alt="Trulli">
-->
<p>This is also visible.</p>

🔍 Hide Inline Content

You can even hide small parts inside one line of HTML:

<p>This <!-- hidden comment --> is a paragraph.</p>

🧪 Debugging Tip

Use comments to comment out parts of your code while troubleshooting layout issues or errors.

❓ Quiz Time

Which of the following is the correct way to add a comment?

  • ❌ <-- This is a comment -->
  • ❌ <## This is a comment ##>
  • ❌ <--> This is a comment </-->
  • ✅ <!-- This is a comment -->

🎯 Summary

  • Comments are written like this: <!-- comment -->
  • They are ignored by the browser
  • Use them to leave notes, hide code, and debug

🛠️ Keep your code clean, organized, and documented with helpful comments!
