Lesson 1: Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It's the language that structures the content of the web. Whether it's a blog post, a photo gallery, or a form to collect user input â€“ HTML is what holds it all together.

HyperText refers to the hyperlinks that connect web pages together. Markup is the set of tags we use to "mark up" content with meaning. It tells the browser how to display things: headings, paragraphs, lists, images, and more.

Basic Structure of an HTML Document

Every HTML document follows this basic template:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

  • <html> â€“ The root element that wraps the entire HTML page.
  • <head> â€“ Contains metadata like the page title, character encoding, styles, and scripts.
  • <body> â€“ Holds the visible content of the page.
