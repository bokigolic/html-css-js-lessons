🔤 What is HTML?

HTML stands for HyperText Markup Language. It’s the basic building block of every web page on the internet.

  • It defines the structure of web content.
  • Uses elements (called tags) to tell the browser how to display content.
  • Examples: headings, paragraphs, links, images, videos, and more.

📄 Basic HTML Page Example

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

🧠 Explanation of the Code

  • <!DOCTYPE html> — tells the browser this is an HTML5 document
  • <html> — the root of the web page
  • <head> — contains metadata like the page’s title
  • <title> — sets the name shown in the browser tab
  • <body> — holds all visible content
  • <h1> — defines a heading (important text)
  • <p> — defines a paragraph

🧩 What is an HTML Element?

An HTML element consists of a start tag, content, and an end tag.

<h1>This is a heading</h1>
<p>>This is a paragraph.</p>

Some elements are empty, meaning they don’t have content or an end tag (e.g., <br> for line break).

🌐 How Browsers Read HTML

Web browsers like Chrome, Firefox, Safari, or Edge don’t show HTML tags. Instead, they use the tags to format and display the content properly on screen.

🧱 Visual Structure of an HTML Page

<html>
  <head>
    <title>Page title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
    <p>This is another paragraph.</p>
  </body>
</html>

👉 Everything inside <body> shows up on the page. The content of <title> appears in the browser’s tab.

📚 Brief History of HTML

  • 1989 — Tim Berners-Lee invents the World Wide Web
  • 1991 — First version of HTML is born
  • 1995 — HTML 2.0 released
  • 1999 — HTML 4.01 becomes standard
  • 2008+ — HTML5 begins and becomes the modern standard

Today, HTML5 is what we use to build modern websites with multimedia, responsive design, and cleaner structure.
