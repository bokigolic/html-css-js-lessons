Lesson 2: Text Formatting

Why Text Formatting Matters

Text formatting in HTML allows you to emphasize important content, improve readability, and structure your text for better user experience. Imagine a page where everything is plain text — it would be hard to scan and boring to read. That’s where formatting tags shine!

💡 Tip: Use formatting tags sparingly and meaningfully to highlight what truly matters.

📝 Common Formatting Tags

  • <b> – Bold (purely visual)
  • <strong> – Important text (semantically meaningful + bold)
  • <i> – Italic (visual)
  • <em> – Emphasized text (semantic + italic)
  • <u> – Underlined text
  • <sup> – Superscript (e.g., x2)
  • <sub> – Subscript (e.g., H2O)
Example: 
<p>This is <b>bold</b>, <i>italic</i>, and <u>underlined</u> text.</p>
<p>Use <strong>strong</strong> and <em>emphasis</em> to convey importance.</p>
<p>Chemical formula: H<sub>2</sub>O | Math: x<sup>2</sup></p>

📏 Line Breaks and Horizontal Rules

HTML provides special tags to help you separate or break up your text:

  • <br> – Line break (like pressing Enter once)
  • <hr> – Horizontal rule (a dividing line)
<p>Line one<br>Line two</p>
<hr>
<p>After the horizontal line.</p>

🎯 Semantic vs. Visual Formatting

There are two types of formatting tags: semantic (carry meaning) and visual (only change appearance).

Visual Semantic
<b> <strong>
<i> <em>

Use semantic tags when possible. They help search engines, screen readers, and developers understand the purpose of the content.

🧠 Try It Yourself

Create your own mini resume using formatting tags:

<h1>Jane Doe</h1>
<p><strong>Web Developer</strong> with a passion for <em>clean code</em> and <u>user experience</u>.</p>
<p>Favorite formula: E = mc<sup>2</sup></p>

📌 Summary

  • Use <b> and <i> for simple bold and italics.
  • Use <strong> and <em> for meaningful emphasis.
  • Use <sup> and <sub> for formulas and notes.
  • Use <br> and <hr> for spacing and separation.
Mini-Challenge: Write a motivational quote and highlight key words with <strong> and <em>.
