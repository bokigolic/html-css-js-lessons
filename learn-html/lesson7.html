Lesson 7: HTML Tables

📊 Why Use Tables?

Tables are useful when you want to display structured data like schedules, price lists, or comparisons in a clean grid format.

🔹 Basic Table Structure

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>28</td>
  </tr>
</table>
  • <table> – creates the table
  • <tr> – table row
  • <th> – table heading (bold & centered by default)
  • <td> – table data (cells)

🧩 colspan & rowspan

These attributes allow you to merge cells across rows or columns:

<td colspan="2">Merged cell</td>
<td rowspan="2">Spans 2 rows</td>

🎨 Styling Tables with CSS

You can make tables look much nicer using CSS:

table {
  border-collapse: collapse;
  width: 100%;
}
th, td {
  border: 1px solid #ccc;
  padding: 10px;
  text-align: left;
}

🧪 Example: Weekly Schedule

<table>
  <tr>
    <th>Day</th>
    <th>Activity</th>
  </tr>
  <tr>
    <td>Monday</td>
    <td>Workout</td>
  </tr>
  <tr>
    <td>Tuesday</td>
    <td>Work on project</td>
  </tr>
</table>

🧠 Practice Challenge

Create a pricing table with 3 columns: Plan, Price, Features

<table>
  <tr>
    <th>Plan</th>
    <th>Price</th>
    <th>Features</th>
  </tr>
  <tr>
    <td>Basic</td>
    <td>$9/month</td>
    <td>Email support</td>
  </tr>
  <tr>
    <td>Pro</td>
    <td>$19/month</td>
    <td>Priority support</td>
  </tr>
</table>

📌 Summary

  • <table> structures your data
  • <tr> for each row, <td> for cells
  • <th> for headers
  • colspan and rowspan help merge cells
  • Use CSS for clean, readable tables
