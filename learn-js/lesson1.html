HTML - Introduction

HTML (HyperText Markup Language) je osnovni jezik za kreiranje web stranica. On opisuje strukturu stranice koristeći oznake, tj. tagove.

HTML dokumenti su zapravo obični tekstualni fajlovi sa ekstenzijom .html i sastoje se od elemenata koji govore browseru kako da prikaže sadržaj.

  • <h1> do <h6> – Naslovi
  • <p> – Paragrafi
  • <a> – Linkovi
  • <img> – Slike

Svaka HTML stranica obično sadrži sledeću strukturu:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
