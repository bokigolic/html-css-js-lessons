🔤 HTML Elements

In HTML, everything is made up of elements. An element tells the browser how to display content — whether it's a heading, a paragraph, an image, or even a line break.

🧱 What is an HTML Element?

Every HTML element consists of:

  • A start tag
  • Some content
  • An end tag (for most elements)

Example structure:

<tagname>Content goes here...</tagname>

📝 Real Examples

<h1>My First Heading</h1>
<p>My first paragraph.</p>
Start Tag Content End Tag
<h1> My First Heading </h1>
<p> My first paragraph </p>
<br> None None

Note: Some tags like <br> don’t need an end tag. These are called empty elements.

📦 Nested HTML Elements

HTML elements can contain other elements. This is called nesting.

For example:

<!DOCTYPE html>
<html>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

🔍 Explanation

  • <html> – The root of the document
  • <body> – Contains all visible content
  • <h1> – Displays a main heading
  • <p> – Displays a paragraph

⚠️ Never Skip the End Tag!

Some browsers will still show your page even if you forget an end tag:

<p>This is a paragraph
<p>This is another paragraph

⚠️ But don’t do this! It can lead to layout bugs and unexpected results.

🕳️ Empty Elements

An empty element doesn’t have any content or a closing tag. A good example is the line break: 

<p>This is a <br> paragraph with a break.</p>

🔠 Are Tags Case Sensitive?

No — HTML is not case sensitive. These are all the same:

<p>Hello</p>
<P>Hello</P>
<P>HELLO</P>

✅ However, the W3C recommends using lowercase for consistency and compatibility with XHTML.
