🌍 Why Responsive?

Responsive design adapts your layout to look great on phones, tablets, and desktops—all using the same HTML.

💡 Tip: Always start mobile-first: design for the smallest screen, then add enhancements.

➡️ Media Query Syntax

@media (min-width: 600px) {
  .container {
    /* styles for tablets and up */
  }
}

You wrap any CSS inside `@media` to apply only on screens wider than 600px.

📌 Summary

  • Start mobile-first: basic styles apply everywhere.
  • Add `@media (min-width: Xpx)` for larger screens.
  • Use flexible units: `%`, `em`, `rem`, `vw`.
  • Test at different window sizes for real responsiveness.
