‚û°Ô∏è What is Flexbox?

CSS Flexbox is a one-dimensional layout system. It helps distribute space and align items in a row or column.

üí° Tip: Use Flexbox for navigation bars, card layouts, and vertical centering!

‚û°Ô∏è Defining a Flex Container

.flex-container {
  display: flex;
  gap: 1rem;
}

This enables horizontal layout with spacing between items.

‚û°Ô∏è Aligning and Justifying

.flex-container {
  justify-content: center; /* align horizontally */
  align-items: center;     /* align vertically */
}

Control how items align within the flex container.

‚û°Ô∏è Wrapping Items

.flex-container {
  flex-wrap: wrap;
}
.flex-item {
  flex: 1 1 150px;
}

This lets items wrap to the next line when space is limited.

üìå Summary

  • `display: flex;` turns a container flexible.
  • `justify-content` = horizontal alignment.
  • `align-items` = vertical alignment.
  • `flex-wrap` allows wrapping of items.
  • `flex: grow shrink basis;` controls item sizing.
