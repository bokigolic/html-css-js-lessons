📋 Why Use Lists?

Lists help structure content into bite-sized chunks: steps in a process, navigation menus, to-dos, or even glossaries. HTML provides multiple types of lists for different purposes.

💡 UX Tip: Lists improve readability and help screen readers navigate structured data.

🔹 Unordered List – <ul>

Used when order doesn’t matter, like a checklist or ingredients:

<ul>
  <li>Milk</li>
  <li>Bread</li>
  <li>Eggs</li>
</ul>
Semantic Tip: Always wrap list items with <li> and keep indentation consistent.

🔸 Ordered List – <ol>

Used when sequence matters, like instructions:

<ol>
  <li>Mix ingredients</li>
  <li>Preheat oven</li>
  <li>Bake for 30 minutes</li>
</ol>

🎨 Customizing List Styles

With CSS, you can change list markers:

ul {
  list-style-type: square;
}
ol {
  list-style-type: upper-roman;
}

Other values include circle, decimal, lower-alpha, etc.

📦 Nested Lists

Lists can contain other lists for hierarchy:

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables</li>
</ul>
🔧 Dev Tip: Always align closing tags properly — nesting errors can cause strange layouts!

📖 Description List – <dl>

Great for glossaries, FAQs, or metadata:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

<dt> = term | <dd> = description

🧠 Practice: Build a Weekly Planner

Use:

  • <ol> for weekday tasks
  • <ul> for weekend fun
  • <dl> for glossary terms
<h2>📅 Weekly Planner</h2>
<ol>
  <li>Monday – Team meeting</li>
  <li>Tuesday – Code review</li>
</ol>
<ul>
  <li>Saturday – Hike⛰️</li>
  <li>Sunday – Chill 🎮</li>
</ul>
<dl>
  <dt>Sprint</dt>
  <dd>2-week development cycle</dd>
</dl>
