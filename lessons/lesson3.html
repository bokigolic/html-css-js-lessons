🧱 HTML Elements – The Core Building Blocks

Every web page is built using HTML elements. An element is a combination of a tag and its content:

<p>This is a paragraph.</p>

Here, <p> is the tag and the entire block is an element. Most elements have an opening and closing tag, although some are self-closing (like <br>).

💡 Tip: Think of HTML elements as Lego bricks. You stack them to build the structure of your page!

📦 Nested Elements

Elements can contain other elements — this is called nesting. Proper nesting ensures your document structure is valid and behaves correctly.

<div>
  <h2>Title</h2>
  <p>This is a <strong>nested</strong> paragraph.</p>
</div>

Always close nested tags in the correct order. Improper nesting can lead to rendering bugs or broken accessibility.

⚠️ Rule: Tags that open last should close first — like stacking and unstacking boxes.

🔑 What Are Attributes?

Attributes add extra info to elements. They're placed inside the opening tag and follow a name="value" format. 

<a href="https://example.com" target="_blank">Visit Example</a>
  • href – Defines the link destination
  • target="_blank" – Opens link in a new tab
👁️‍🗨️ FYI: Attributes enhance functionality but don’t usually appear on the screen — they work behind the scenes.

🌐 Global Attributes

These attributes can be used on most HTML elements to control behavior or styling:

  • id – Uniquely identifies the element
  • class – Categorizes elements for styling or scripting
  • style – Inline CSS styling
  • title – Tooltip shown on hover
<p id="intro" class="highlight" style="color:blue" title="Intro paragraph">
  Welcome to my page!
</p>

🔍 Why Semantic Tags Matter

Semantic elements (like <article>, <section>, <header>) describe their content clearly, which improves:

  • Accessibility (for screen readers)
  • SEO (for search engines)
  • Code readability (for developers)
💡 Tip: Avoid using <div> for everything. Use semantic tags wherever possible.

🧪 Practice Activity

Create a basic profile card using elements and attributes:

<div class="card">
  <img src="cat.jpg" alt="A cute cat looking curious" />
  <h3>Meet Miso</h3>
  <p class="caption" title="This is Miso the cat">Curious and cuddly!</p>
</div>

📌 Summary

  • Elements structure your content — use them wisely
  • Tags can nest, but they must be properly closed
  • Attributes enhance element functionality
  • Use semantic tags and global attributes like id, class, title
