🤝 What is Flexbox?

Flexbox helps you align and distribute space among items in a container — ideal for one-dimensional layouts.

💡 Tip: Use Flexbox for rows or columns, not complex grids (that’s Grid’s job!).

➡️ Setting Up Flex Container

.container {
  display: flex;
  flex-direction: row; /* or column */
  gap: 1rem;
}

`flex-direction` defines the main axis. `gap` adds space between flex items.

UX Tip: Combine `justify-content` and `align-items` to center elements.

➡️ Key Flex Properties

  • justify-content: center | space-between | space-around; – align along main axis
  • align-items: center | flex-start | stretch; – align along cross axis
  • flex: 1; – make items grow equally

📌 Summary

  • Enable flex with `display: flex`.
  • Define direction with `flex-direction`.
  • Use `justify-content` and `align-items` to position items.
  • `flex` allows items to grow/shrink proportionally.
