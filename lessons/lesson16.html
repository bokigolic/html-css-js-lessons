📦 What Are CSS Variables?

CSS variables (custom properties) let you store values once and reuse them, making theming and maintenance easy.

💡 Tip: Define variables in :root for global usage.

➡️ Defining Variables

:root {
  --primary: #0d6efd;
  --secondary: #6c757d;
  --background: #ffffff;
  --text: #212529;
}

Use them with var(--primary) inside your CSS.

➡️ Using Variables

body {
  background-color: var(--background);
  color: var(--text);
}
button {
  background-color: var(--primary);
  color: white;
}
UX Tip: Changing a single variable updates the whole theme.
