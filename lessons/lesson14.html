🌈 What is CSS?

CSS (Cascading Style Sheets) controls the appearance of HTML elements—color, layout, spacing, fonts and more. It makes your content visually engaging!

💡 Tip: Think of CSS as the “clothes” and “make-up” for your HTML “skeleton”.

🛠️ CSS Selectors

Select which HTML elements you want to style:

/* element */ 
h1 { color: darkblue; }

/* class */ 
.tip-box { background: lightgoldenrodyellow; }

/* id */ 
#special { font-weight: bold; }
💡 Pro Tip: Classes are reusable; IDs are unique per page.

🎯 Common Properties

  • color – text color
  • background-color
  • font-size
  • margin, padding
  • border
p {
  color: #333;
  background-color: #f0f0f0;
  margin: 1rem 0;
  padding: 0.5rem;
  border: 1px solid #ccc;
}

📦 The Box Model

Every HTML element has padding, border, margin around its content:

div {
  width: 200px;
  padding: 10px;
  border: 2px solid red;
  margin: 20px;
}
💡 Tip: Use box-sizing: border-box; to simplify width calculations.
