🗺️ What Is CSS Grid?

CSS Grid lets you create complex two-dimensional layouts with rows and columns—perfect for modern design.

💡 Tip: Think of grid like a spreadsheet—you define rows and columns, then place items.

➡️ Basic Grid Setup

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

Use `1fr` to divide available space equally; `gap` controls spacing between items.
