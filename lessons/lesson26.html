ğŸ”‘ What Are CSS Variables?

CSS variables (custom properties) store reusable values for colors, spacing, fonts, and more. They make your code DRY and enable theming.

ğŸ’¡ Tip: Use `--` prefix, and reference with `var(--name)` everywhere!

â¡ï¸ Define & Use Variables

:root {
  --primary-color: #0d6efd;
  --bg-light: #f8f9fa;
}
.button {
  background: var(--primary-color);
  color: white;
}

â¡ï¸ Theme Switcher

Use JS to toggle between light/dark by updating custom properties:

const root = document.documentElement;
document.querySelector('.toggle-theme').onclick = () => {
  root.classList.toggle('dark-mode');
  root.style.setProperty('--primary-color', root.classList.contains('dark-mode') ? '#66b2ff' : '#0d6efd');
};
