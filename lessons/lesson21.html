🧩 What Are CSS Variables?

CSS variables (custom properties) let you store values in `--your-var` and reuse them across your stylesheet.

💡 Tip: Think of variables like global theme constants—change once, update everywhere!

➡️ Declaring & Using Variables

:root {
  --main-color: #66b2ff;
}
.button {
  background: var(--main-color);
}

You declare a variable in `:root` and access it with `var(--var-name)`.

➡️ Variable Scope

.card {
  --card-padding: 1.5rem;
  padding: var(--card-padding);
}

Variables respect CSS scope—declaring inside a selector makes it local to its descendants.

🕺 What Are CSS Animations?

Animations let you change CSS properties smoothly over time using `@keyframes` and `animation:` rules.

💡 Tip: Use animations to draw attention—but don’t go overboard!

📌 Summary

  • CSS variables start with `--` and are retrieved with `var()`.
  • Declare them in `:root` for global or inside selectors for scoped use.
  • `@keyframes` defines animation steps; `animation:` applies them.
  • You can control duration, easing, delay, iteration, and direction.
