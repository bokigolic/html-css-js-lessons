🖋️ Why Text Formatting Matters

Text formatting allows you to highlight key points, organize content for better readability, and add visual interest to plain text. Without formatting, your content would feel flat, overwhelming, and hard to scan.

HTML offers a powerful set of tags for formatting text — both for visual style and semantic meaning. The better you understand these tools, the more effectively you’ll communicate with users and assistive technologies.

💡 Tip: Always format with intention. Don’t use <b> just to make something look "cool" — use tags that match the meaning of your content.

📝 Common Formatting Tags

  • <b> – Bold (visual only)
  • <strong> – Important (semantic + bold)
  • <i> – Italic (visual only)
  • <em> – Emphasized (semantic + italic)
  • <u> – Underline
  • <sup> – Superscript (e.g. x2)
  • <sub> – Subscript (e.g. H2O)
📘 Example: 
<p>This is <b>bold</b>, <i>italic</i>, and <u>underlined</u> text.</p>
<p>Use <strong>strong</strong> and <em>emphasis</em> to convey importance.</p>
<p>Formula: H<sub>2</sub>O | Math: x<sup>2</sup></p>

📏 Line Breaks and Horizontal Rules

Need to break up text or add spacing? Use these tags:

  • <br> – Adds a line break (like hitting “Enter”)
  • <hr> – Adds a horizontal dividing line
<p>First line<br>Second line</p>
<hr>
<p>After horizontal rule.</p>

🎯 Semantic vs. Visual Formatting

Not all tags are equal. Some are semantic — they convey meaning. Others are just visual — they only affect how content looks.

Visual Semantic
<b> <strong>
<i> <em>
🧠 Best Practice: Use <strong> and <em> for content that’s meaningful, like alerts, warnings, and emphasis in instructions.

🧠 Practice Challenge

Create a simple resume-style layout using formatting tags:

<h1>Jane Doe</h1>
<p><strong>Web Developer</strong> with a passion for <em>clean code</em> and <u>UX</u>.</p>
<p>Quote: <em>"Code is poetry"</em></p>
<p>Math: E = mc<sup>2</sup></p>

📌 Summary

  • Use <strong> and <em> for semantic emphasis
  • Use <b>, <i> for visual-only changes (if necessary)
  • <br> and <hr> help break up content and improve flow
  • Text formatting boosts readability, hierarchy, and UX
