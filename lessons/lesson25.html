💡 Why Animations?

Animations make interfaces feel alive: they guide users, provide feedback, and add polish to experiences.

UX Tip: Keep animations smooth and purposeful — don't distract users!

➡️ Keyframes Animation

@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.2); opacity: 0.7; }
}

.animate-box {
  animation: pulse 2s ease-in-out infinite;
}

That box pulses smoothly — great for drawing attention to buttons or cards.

➡️ CSS Transitions

.transition-box {
  transition: transform 0.3s ease, background 0.3s;
}
.transition-box:hover {
  transform: rotate(15deg) scale(1.1);
  background: var(--primary);
}

Hover to see a smooth rotate and color change — perfect for buttons or interactive elements.

📌 Summary

  • Use `@keyframes` + `animation` for repeating motions.
  • Use `transition` for smooth style changes on interaction.
  • Keep animations intentional, not overwhelming.
