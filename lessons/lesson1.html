🌐 What is HTML?

HTML (HyperText Markup Language) is the 🧱 of the web—it structures text, links, images, and more.

💡 Tip: Think of HTML as the blueprint of a building—the foundation for everything else!

🧩 Basic Document Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is HTML!</p>
  </body>
</html>

This template includes:

  • <!DOCTYPE html> – declares HTML5
  • <html> – root element
  • <head> – meta-information (title, styles)
  • <body> – visible content
Example: 
<h1>Welcome!</h1>
<p>Start your HTML journey here.</p>
⚠️ Best Practices: Always indent nested tags—clean, readable code saves time later!

🧪 Try This:

  • Two paragraphs describing yourself
  • A comment explaining one section

📌 Summary

  • HTML structures web content.
  • Start with the standard boilerplate template.
  • Use semantic tags (<h1>, <p>, etc.).
