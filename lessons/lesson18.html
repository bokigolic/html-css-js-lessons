🌐 What is Responsive Design?

Responsive design ensures your website looks and works great on any screen, from phones to desktops.

💡 Tip: Always start from mobile-first—style for small screens first, then expand!

➡️ Using Media Queries

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
@media (min-width: 601px) {
  body {
    font-size: 18px;
  }
}

➡️ Fluid Layout Example

.container {
  display: flex;
  gap: 1rem;
}
@media (max-width: 800px) {
  .container {
    flex-direction: column;
  }
}

📌 Summary

  • Use `@media` queries to adapt styles responsively.
  • Prefer a mobile-first approach with `max-width` breakpoints.
  • Use flexible layouts (flex/grid) and fluid sizes (%, rem).
