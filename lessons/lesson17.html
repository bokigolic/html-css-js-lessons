🧩 Why Layout Matters

Flexbox and Grid are modern CSS layout models that help you build responsive and efficient layouts without hacks.

💡 Tip: Use Flexbox for 1D layouts (rows or columns), Grid for complex 2D structures.

➡️ Flexbox Essentials

Flexbox makes it easy to align, distribute space, and reorder items:

.container {
  display: flex;
  flex-direction: row; /* or column */
  justify-content: space-between;
  align-items: center;
}
💡 Tip: Use `justify-content` for distribution, `align-items` for cross-axis alignment.

➡️ Grid Essentials

Grid gives you full control over rows and columns:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 1rem;
}
💡 Tip: Use `repeat`, `fr`, and `grid-gap` for flexible layouts.
