📐 Why CSS Grid?

CSS Grid lets you build flexible, two-dimensional layouts easily — perfect for cards, galleries, or full-page layouts.

UX Tip: Mobile-first → let items wrap naturally with `auto-fit`.

➡️ Grid Container & Items

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

Three equal columns, items flow row by row with consistent gaps.

➡️ Responsive with `auto-fit`

grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

Grid adapts to screen size — columns adjust automatically!

➡️ Placing Items Manually

.item-a {
  grid-column: 1 / span 2;
  grid-row: 1 / 3;
}

Make elements span multiple rows/columns for flexible layouts.

📌 Summary

  • `display: grid` to create container
  • Use `grid-template-columns` & `gap` for layout control
  • `auto-fit` + `minmax` = responsive grids
  • Use `grid-column` / `grid-row` for custom placement
